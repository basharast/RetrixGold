VFS Layer (New)

This means you can use Libretro cores with RetriXGold directly

the new VFS layer is much smarter and provides compatiblity with archived roms (even if the core doesn't support that)

it can also solve many issues that prevent UWP app from accessing to the content

Easy to use (New)

Each core will have it's own games and system folders

Forget to add everything in one place

with the new virtual files integration the core will be redirected by default to it's own folders

Important

To avoid conflict with the old releases this will be installed as 'RetriXGold 3' and will not update the current app

You can backup the data from settings and restore it in the new app
